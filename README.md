# TinyApp project

Tiny app is a web application built with node that will allow a user to login/logout and shorten and keep a number of URLs.

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting started

- Install all dependencies
- Run the server using command node express_server.js
